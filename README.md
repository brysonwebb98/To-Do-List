# Development Environment

For this project I used Visual Studio Code as my development environment. I wrote the application in JavaScript, along with HTML and CSS to handle the layout and structure of the page. I also included the Luxon library through a CDN link so I could use it directly in the browser without extra setup.

The project runs entirely client-side, using JavaScript to create elements, update the task list, and filter tasks in real time. Building it in VS Code made it easy to test changes quickly and keep my files organized as I worked.
